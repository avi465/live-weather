
# Live Weather

This is a simple live weather app made using openweather map api.


## Screenshots

![App Screenshot](https://i.imgur.com/XqQXQZb.png)


## Run Locally

Clone the project

```bash
git clone https://github.com/avi465/live-weather.git
```

Go to the project directory

```bash
cd my-project
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run start
```
